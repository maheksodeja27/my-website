## 📄 Pages Description

### 🏠 Home (index.html)
- Introduction to the website  
- Navigation menu  
- Overview of features  

### 👤 About (about.html)
- Information about the team  
- Mission and vision  

### 🛠️ Services (services.html)
- List of services offered  
- Displayed using an HTML table  

### 📝 Blog (blog.html)
- Sample blog posts  
- Basic content structure  

### 📞 Contact (contact.html)
- Contact form  
- User can enter name, email, and message  

---

## 🚀 How to Run the Project

1. Create a folder named `my-website`
2. Add all HTML files inside the folder
3. Open `index.html` in any web browser
4. Use the navigation links to explore different pages
